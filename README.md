# danils.xyz

My new personal blog based on Astro. Based on [Astro Micro](https://astro-micro.vercel.app/). 
Includes MDX-based blog, tags page, library page, and site-wide search.

## Development
1. Clone repo
2. Install dependencies:
   ```bash
   npm install
   ```
4. Run development server:
   ```bash
   npm run dev
   ```
